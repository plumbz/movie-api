Movie Api Documentation.

Welcome to the documentation of Movie API. This API allows the user to recieve information about movies, directors' Bio, and genres. The Movie API allows users to create a profile to save a list of thier favorite movies as well as to delete them. This API is built using Node.js, Express, Body- Parser and UUI.
