Movie Api Documentation.

Welcome to the documentation of Movie API. This API allows the user to recieve information about movies, directors' Bio, and genres. The Movie API allows users to create a profile to save a list of thier favorite movies as well as to delete them. This API is built using Node.js, Express, Body- Parser and UUI.

Request URL Description HTTP Method Query Parameters Request Body Response body
Get All Movies /movies Returns a list of all movies in the database. GET None None JSON array of movie objects
Get Movie by Title /movies/:title Returns data of the specific movie being searched. GET title (URL parameter) None JSON object holding movie details (title, description, genre, director, image URL, etc.)
Get Genre by Name /genres/:name Returns data about a genre specified by it`s name. GET name (URL parameter) None JSON object with genre details (name, description)
Get Director by Name /directors/:name Returns data about a director specified by their name. GET name (URL parameter) None JSON object holding director details (name, birth year, death year)
Create User /users Added new user. POST None JSON object of the new user Confirmation message or error message
Update User /users/:id Allows a user to update their personal information. PUT id (URL parameter) JSON object with updated user data (username, password, email, dateOfBirth) Confirmation message
Add Movie to Favorites /users/:id/movieTitle Add a movie to a user's list of favorites. POST id and movie title (URL parameter) None Updated list of favorite movies or confirmation message
Remove Movie from Favorites /users/:id/movieTitle Remove a movie from a user's list of favorites. DELETE id and movie title (URL parameter) None Updated list of favorite movies or confirmation message
Deregister User /users/:id Allows an existing user to deregister. DELETE id (URL parameter) None Confirmation message or error message
